# Sistema de Gerenciamento de Acervos - Museu Treze de Maio

## Sobre o Projeto

Este projeto consiste no desenvolvimento de um sistema de gerenciamento de acervos para o **Museu Treze de Maio**, uma instituição dedicada à preservação e valorização da cultura afro-brasileira em Santa Maria, Rio Grande do Sul.

O museu possui um vasto patrimônio histórico e cultural que necessita ser digitalizado, catalogado e disponibilizado de forma acessível tanto para a equipe administrativa quanto para pesquisadores e o público em geral.

## Objetivo

Desenvolver uma solução integrada de software que permita a catalogação, o gerenciamento e a consulta pública dos acervos do museu, utilizando SQL Server como sistema gerenciador de banco de dados.

## Acervos Gerenciados

O sistema foi projetado para gerenciar dois acervos distintos:

### 1. Acervo Histórico
Composto por materiais diversos relacionados à história e cultura afro-brasileira:
- Jornais históricos
- Atas de reuniões e documentos oficiais
- Fotografias
- Cartas e correspondências
- Relatos orais
- Objetos e artefatos

### 2. Biblioteca Comunitária
Acervo bibliográfico estruturado contendo:
- Livros
- Revistas
- Jornais

## Perfis de Usuários

O sistema atende a diferentes perfis de usuários com necessidades específicas:

- **Equipe do Museu (Administradores)**: Realizam a catalogação, edição e gerenciamento completo dos itens do acervo
- **Público em Geral**: Podem realizar consultas e pesquisas para descobrir e conhecer o acervo disponível

## Importância

Este sistema é fundamental para:
- Preservar a memória histórica e cultural afro-brasileira
- Facilitar o acesso à informação para pesquisadores
- Democratizar o conhecimento para a comunidade
- Garantir a organização e proteção do patrimônio do museu
- Possibilitar a digitalização e catalogação adequada dos materiais
